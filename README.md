# Campus Connect

Campus Connect is a full-stack platform designed to improve student engagement by helping users discover clubs, explore events, connect with campus communities, and participate in university activities through a centralized digital experience.

Developed as a practical full-stack project, Campus Connect explores how modern web technologies can simplify event discovery, community building, and student interaction within a campus ecosystem.

## Overview

Campus Connect provides a unified platform where students can:

- Discover campus clubs and organizations
- Explore upcoming events
- Join and participate in activities
- Connect with student communities
- Interact through club-focused experiences

The project combines a modern frontend experience with backend services to support scalable student engagement workflows.

## Features

- Club discovery and browsing
- Event management and exploration
- Individual event pages
- Event participation workflows
- Club interaction and communication features
- Authentication and user management
- Real-time interaction support
- Media upload support

## Getting Started

### Clone the Repository

```bash
git clone <repository-url>
cd CampusConnect-Project
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
